# 중앙 AI 설정(.ai) 안내

이 폴더는 코드 도우미/AI 도구들이 일관된 맥락과 가이드를 공유하도록 하는 중앙 설정 위치입니다.  
아래 문서를 우선적으로 참조해 개발 보조 도구 세팅, 코드 생성, 리뷰 품질을 향상시킵니다.

- architecture.md: 아키텍처/흐름(라우팅/가드, 상태/데이터, API 레이어)
- coding_standards.md: 코딩 표준(네이밍, 타입, 리액트/TS, 에러 처리 등)
- seo.md: SEO 개선 작업

권장 사용 방법

- 자동화/AI 도구(Copilot, Cursor, Claude 등)는 이 폴더를 먼저 읽고 응답에 반영합니다.
- 레포에 큰 변경이 있을 때는 본 문서들을 최신화해 싱크를 유지합니다.
- 팀 합의가 필요한 변경은 PR로 제안 후 합의되면 문서화합니다.

문서 업데이트 원칙

- 실제 코드베이스와 불일치하지 않도록 최소 주기(주 1회 이상)로 점검
- 합의되지 않은 방향/실험은 분리 기록(주석/별도 섹션) 후 명확히 표시
