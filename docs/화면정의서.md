## 1. 프로젝트 개요

### 1.1 서비스 소개
"뭐입지"는 AI 기반 코디 추천 서비스입니다. 사용자가 4가지 간단한 질문에 답하면 AI가 무신사 상품을 기반으로 최적의 코디를 추천합니다.

### 1.2 핵심 기능
- 4가지 질문 기반 스타일 분석
- AI 코디 추천 (상의, 하의, 신발)
- 무신사 연동 구매
- 코디 저장 및 관리
- 구독 서비스 (Basic/Pro)

### 1.3 기술 스택
- **Frontend:** React 19, TypeScript, Tailwind CSS
- **Router:** React Router DOM
- **Icons:** Lucide React
- **Build:** Vite

### 3.1 메인 홈 화면 (HOME)

**화면 ID:** HOME
**경로:** `/`
**스크린샷:** `screenshots/01_Home.png`, `screenshots/01_Home_Desktop.png`

#### 화면 설명
서비스의 메인 랜딩 페이지로, 서비스 소개와 시작 버튼을 제공합니다.

#### 화면 구성 요소

| No. | 요소명 | 타입 | 설명 |
|-----|--------|------|------|
| 1 | 네비게이션 바 | Header | 로고, 로그인/마이페이지 버튼 |
| 2 | 히어로 섹션 | Section | 메인 타이틀, 설명, CTA 버튼 |
| 3 | 무료 체험 배너 | Banner | "5회 무료 체험" 안내 |
| 4 | 4 Questions 섹션 | Section | 서비스 진행 방식 설명 |
| 5 | 푸터 | Footer | 서비스 정보 |

#### 주요 기능
- **코디 추천 받기 버튼:** `/quiz` 페이지로 이동
- **무료로 시작하기 버튼:** `/signup` 페이지로 이동
- **로그인 버튼:** `/login` 페이지로 이동 (비로그인 시)
- **마이페이지 버튼:** `/mypage` 페이지로 이동 (로그인 시)

#### 반응형 처리
- Mobile: 단일 컬럼 레이아웃
- Desktop: 2컬럼 그리드 레이아웃

**화면 ID:** QUIZ
**경로:** `/quiz`
**스크린샷:** `screenshots/04_Quiz_Q1.png`

#### 화면 설명
4가지 질문을 통해 사용자의 스타일 선호도를 파악합니다.

#### 질문 목록

| Q | 질문 | 선택지 |
|---|------|--------|
| Q1 | 어디 가요? | 학교, 회사, 데이트, 외출 |
| Q2 | 어떻게 보이고 싶어요? | 깔끔하게, 편하게, 멋있게, 독특하게 |
| Q3 | 체형 고민? | 없음, 배, 마른편, 키 |
| Q4 | 예산은? | 10만원, 20만원, 30만원, 무관 |

#### 화면 구성 요소

| No. | 요소명 | 타입 | 설명 |
|-----|--------|------|------|
| 1 | 뒤로가기 버튼 | Button | 이전 질문/홈으로 이동 |
| 2 | 진행률 바 | ProgressBar | 현재 진행 상태 (1/4 ~ 4/4) |
| 3 | 질문 번호 | Text | "Q1 / 4" 형식 |
| 4 | 질문 텍스트 | Text | 현재 질문 |
| 5 | 선택지 버튼 | Button Grid | 2x2 그리드 선택지 |
| 6 | 다음 버튼 | Button | 다음 질문/결과 보기 |

#### 상태 관리
- 선택한 답변은 상태로 저장
- 마지막 질문 후 `/loading`으로 이동
